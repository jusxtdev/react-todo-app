# Todo App

A small React todo app built with Vite. It lets users add tasks with a title and description, mark tasks as complete, and delete tasks from the list.

The UI follows a dark teal and coral theme inspired by the provided AngelList reference.

## Built With

- React
- Vite
- Tailwind CSS

## Work Split

I built the main React structure, todo data flow, and app functionality.

AI help was used for:
- Tailwind styling for the app background and components
- Matching the visual theme
- Writing this README
